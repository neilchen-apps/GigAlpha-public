# GigAlpha 戰情室 (GigAlpha)

## 核心功能

- 精準利潤計算：扣除油錢、保養等隱形成本，精算真實時薪與每單淨利潤
- 懸浮工具箱快記：3 秒記一筆，上班打卡、完成外送、油錢支出隨手記
- 圖表化數據分析：依時間趨勢、消費種類、記帳區塊自由切換圖表
- 個人雲端備份：透過 Google Drive 帳號備份與還原資料，資料仍由你個人掌控
- 極致隱私保護：精確 GPS 與財務數據皆在地端運算，絕不存入開發者伺服器
